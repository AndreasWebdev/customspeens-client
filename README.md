## Compilation Instructions (Windows)
1. Clone the repository.
2. Install [node.js](https://nodejs.org/)
3. Open the command prompt to install the dependencies ```npm install```
6. Run ```npm run start```

### Package Electron App
6. Run ```npm run build```
